AC to DC Converter – PCB Design Project
This project focuses on designing and fabricating a reliable AC to DC power conversion circuit capable of converting 230V AC mains supply into regulated DC output.

Features:
Step-down transformer / SMPS topology
Bridge rectifier for AC to DC conversion
Capacitor filtering stage for ripple reduction
Voltage regulation using IC regulator (e.g., 7805 / 7812 / LM317)
Overcurrent and short circuit protection
Compact PCB layout designed in KiCad/Altium/Eagle

Tools Used:
KiCad / Eagle / Altium Designer
Multimeter & Oscilloscope for testing
Soldering & PCB fabrication process

Applications:
Embedded systems power supply
IoT devices
Microcontroller-based projects
Robotics circuits
